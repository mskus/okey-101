# 🃏 101 Okey Pro Canlı - Gelişmiş Skor & Hesap Takip Uygulaması

**Canlı Test Linki:** [https://mskus.github.io/okey-101/](https://mskus.github.io/okey-101/)

Bu proje, karmaşık 101 Okey kurallarını dijital ortamda hatasız hesaplamak için tasarlanmıştır. Yeni eklenen **Firebase Realtime Database** entegrasyonu sayesinde masadaki herkes uygulamayı kendi cihazından açıp, anlık olarak senkronize şekilde skorları takip edebilir.

## ✨ Öne Çıkan Özellikler

* **Canlı Masa Senkronizasyonu:** Firebase entegrasyonu ile bir kişi skoru girdiğinde tüm masanın ekranı saniyeler içinde güncellenir.
* **Gelişmiş Kural Motoru & Otomasyon:** Normal bitiş, Elden bitiş (x2), Jokerle bitiş (x2) ve Elden+Joker bitiş (x4) gibi çarpanlar otomatik işler. Biten kişi ve bitiş türü seçildiği an tabloya **otomatik aktarılır**.
* **Dinamik Takım Renkleri:** Eşli oyunda "Takım A" ve "Takım B" renkleri (Pastel Mavi ve Kırmızı) arşive, tablolara ve bitiş butonlarına kadar tüm UI'a kusursuz yansıtılmıştır.
* **Akıllı Çekmece Sistemi:** Toplam skor tablosu ve **entegre hesap makinesi** oyun alanını daraltmamak için ekranın altında çekmece olarak gizlenmiştir.
* **Gelişmiş Arşiv & İstatistikler:** Maçlar sıfırlandığında yerel hafızaya alınır. Hangi oyuncunun/takımın toplam kaç el bittiği ve yediği toplam cezalar istatistik olarak listelenir.

## 🛠️ Kullanılan Teknolojiler
* **Frontend:** HTML5, CSS3, Vanilla JavaScript (Modern State Management)
* **Backend / Database:** Firebase Realtime Database (BaaS)
* **Hosting:** GitHub Pages
